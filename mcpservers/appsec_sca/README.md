# Application Security SCA MCP Server

MCP server providing Software Composition Analysis tools:
- **Syft**: SBOM generation
- **Dependency-Check**: OWASP dependency vulnerability scanner
- **Retire.js**: JavaScript library vulnerability scanner

## Tools

- `syft_scan_repository(repo_url, ...)`
- `dependency_check_scan_repository(repo_url, ...)`
- `retire_js_scan_repository(repo_url, ...)`
