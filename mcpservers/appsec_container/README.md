# Application Security Container MCP Server

MCP server providing Container Security tools:
- **Grype**: Container vulnerability scanner
- **Trivy**: Comprehensive container scanner

## Tools

- `grype_scan_container(image_name, ...)`
- `grype_scan_repository(repo_url, ...)`
- `trivy_scan_container(image_name, ...)`
- `trivy_scan_repository(repo_url, ...)`
