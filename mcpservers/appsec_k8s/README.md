# Application Security Kubernetes MCP Server

MCP server providing Kubernetes Security tools:
- **Kubescape**: K8s security and compliance
- **Kube-Bench**: Kubernetes security compliance scanner
- **Kube-Hunter**: Kubernetes cluster vulnerability scanner

## Tools

- `kubescape_scan_repository(repo_url, ...)`
- `kube_bench_scan_repository(repo_url, ...)`
- `kube_hunter_scan_repository(repo_url, ...)`
