# Application Security Secrets MCP Server

MCP server providing Secret Scanning tools:
- **Gitleaks**: Secret detection in repositories
- **TruffleHog**: Secret scanning with verification
- **GitGuardian**: Enterprise secret scanning (200+ secret types)

## Tools

- `gitleaks_scan_repository(repo_url, ...)`
- `trufflehog_scan_repository(repo_url, ...)`
- `gitguardian_scan_repository(repo_url, ...)`
